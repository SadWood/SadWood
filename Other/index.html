# Mac中如何运行.sh文件

.sh文件也就是shell脚本，Mac下如何运行，有两种方式：

  1. 直接讲文件拖入终端
  2. 直接在终端中运行，进入文件所在目录，运行./***.sh

如下：

重要

如果出现permission denied:./***.sh的提示，说明是权限问题，需要设置一下权限

chmod 777 ***.sh

设置完成后，再运行即可
