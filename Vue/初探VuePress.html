# 1.1、最后更新时间

在VuePress中的有一个配置项是在MarkDown末尾添加最后更新时间，在官方文档中是这样介绍的：

你可以通过 themeConfig.lastUpdated 选项来获取每个文件最后一次 git 提交的 UNIX 时间戳(ms)，同时它将以合适的日期格式显示在每一页的底部：

module.exports = {
  themeConfig: {
    lastUpdated: 'Last Updated', // string | boolean
  }
}

请注意，themeConfig.lastUpdated 默认是关闭的，如果给定一个字符串，它将会作为前缀显示（默认值是：Last Updated）。

使用须知

由于 lastUpdated 是基于 git 的, 所以你只能在一个基于 git 的项目中启用它。

但是尝试后，发现现实格式如下：Last Updated: 7/31/2018, 11:58:34 PM

其中Last Updated是配置中所定义的，至于时间格式，一开始猜测是根据git log中的时间格式来显示的

但是VuePress的中文文档末尾的格式不同上次更新: 2018/7/31 下午11:58:34

尝试使用git config log.date local修改git log中的时间显示，重新编译部署后显示依旧
