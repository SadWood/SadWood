# Mac OS 终端利器 iTerm2

很多时候，我们在使用系统自带的终端，总觉得有些不太方便，再加上最近接触了win下的cmder，就在思考Mac OS下是否有同样优秀的终端，于是就找到了ITerm2，根据网上的诸多配置以及资料，发现了一些过时或者没有说的特别清楚的地方，所以，把我走过的路（遇到的坑）都记录下来。

# 一、安装与设置

# 1、安装iTerm2

官方下载地址：https://www.iterm2.com/downloads.html

下载后是一个压缩包，然后解压后是可执行文件iTerm.app，双击安装，或者直接拖入Application目录下。

或者使用Homebrew进行安装：

brew cask install iterm2

提示

这里的brew是Homebrew需要另外安装，具体的安装过程可以百度/Google，后面相关内容也有部分使用了Homebrew

# 2、设置iTerm2为默认终端

iTerm2 ==> Make iTerm2 Default Term

确保最终如图所示：

# 3、设置显示/隐藏快捷键

iTerm2 ==> Preferences ==> Keys ==> Hotkey

设置如下，讲command+.（或者任意无冲组合键）设置为显示/隐藏iTerm的快捷键，方便快速打开iTerm2

# 二、主题

# 1、配色

我使用的配色方案是material-design-colors

直接打开终端(Terminal)，复制黏贴如下：

curl -O https://raw.githubusercontent.com/MartinSeeler/iterm2-material-design/master/material-design-colors.itermcolors

接着进行如下设置：

iTerm2 ==> Preference ==> Profiles ==> Colors ==> Color Presets ==> Import

导入刚刚下载的material-design-colors.itermcolors文件，最后选择material-design-colors即可

# 2、Oh My Zsh主题

Oh My Zsh就如同他的官方介绍一样：

Your terminal never felt this good before.

官网地址：https://ohmyz.sh/

官方提供两种命令安装方式，我们选择第一种curl命令：

sh -c "(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

然后需要编辑.zshrc文件，由于.***文件在Mac OS中属于隐藏文件，可以使用vim进行编辑（如何使用自行百度），需要在终端输入vi ~/.zshrc，这里使用的是比较常用的 zsh 主题之一agnoster，我们需要修改Zsh主题为ZSH_THEME="agnoster"，如若不会使用vim，也可以通过编辑器，如：VS Code对.zshrc文件进行修改操作，之后执行source ~/.zshrc，重新加载.zshrc。

除了agnoster，也可以根据个人喜欢，选择自己心意的主题。

zsh的主题列表：https://github.com/robbyrussell/oh-my-zsh/wiki/themes

# 三、插件

# 1、语法高亮

使用 Homebrew 安装：

brew install zsh-syntax-highlighting

依旧是编辑.zshrc文件，在最后一行加上后，执行source ~/.zshrc，：

source /usr/local/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

# 四、技巧

# 1、隐藏用户名和主机名

如上设置完成后，会发现会显示用户名和主机名，特别长，也不好看，如：xxxxxx@xxxxxxdeMacBook-Pro，那么我们可以手动修改主题文件，来进行个性化的设置。

找到主题文件，在~/.oh-my-zsh/themes下，如我选用的主题是agnoster，那么就找到并打开agnoster.zsh-theme，找到如下设置：

想要完全取消context的显示，可以直接将83行注释

prompt_context() {
  if [[ "$USER" != "$DEFAULT_USER" || -n "$SSH_CLIENT" ]]; then
    # prompt_segment black default "%(!.%{%F{yellow}%}.)$USER@%m"
  fi
}

或者自行修改$USER@%m部分，进行个性化设置。

# 2、临时代理配置

需要自行翻墙，然后根据APP设置的监听端口号自行修改port值，如port为1087，设置如下：

export http_proxy=http://127.0.0.1:1087;
export https_proxy=http://127.0.0.1:1087;

通过curl ip.cn查看是否设置成功

注意

临时代理配置仅针对当前iTerm标签页，关闭iTerm重新打开或者在新标签页中，均需要重新设置。

# 3、左右键跳转

主要是按住option + → or ←键，在命令的开始和结尾跳转切换，原本是不生效的，需要手动开启。

Profiles ==> Keys ==> Load Preset...==> Natural Text Editing 即可

# 4、dash&zsh切换

chsh -s /bin/zsh	#切换为zsh
chsh -s /bin/bash	#切换为bash

# 五、坑

# 1、错误提示：zsh: command not found mysql

一开始看到这个错误提示的时候，一脸懵逼，因为在没有安装iTerm2之前MySQL已经安装好，没有任何问题，但是这里却提示相关命令没找到。

经过一阵搜索后得知原因：bash shell 以及zsh shell 是两种读取系统环境变量（使用mysql命令的前提是，肯定已经在bash的.bash_profile已经配置相关的环境变量了，从而才能使用mysql命令）。

那么解决方法就是将bash中的环境配置导入.zshrc中，保存后执行source ~/.zshrc：

# User configuration

source ~/.bash_profile

最后MySQL命令就可以正常使用了。
